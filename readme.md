## ✨ Core Features

WiseCart comes packed with features designed to enhance your shopping experience:

-   🔍 **Advanced Product Search:** Quickly find any product you're looking for with our powerful and intuitive search engine.
-   ↔️ **Side-by-Side Comparison:** Easily compare product specifications, prices, and seller ratings from various online retailers in one place.
-   📉 **Real-Time Price Tracking:** Get instant alerts and historical data on price drops, ensuring you never miss a great deal.
-   👤 **Personalized User Profiles:** Create your own profile to save your preferences, track your favorite items, and manage your shopping lists.
-   🛍️ **Verified Shop Information:** Access detailed information and genuine user ratings for a multitude of online shops.
-   📱 **Fully Responsive Design:** Enjoy a seamless experience on any device, thanks to our modern and clean interface built with **Tailwind CSS**.

## ⚡ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have the following installed on your machine:

* Node.js (v14 or higher)
* npm or yarn
* Git

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/wisecart.git](https://github.com/your-username/wisecart.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd wisecart
    ```
3.  **Install NPM packages:**
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```
    or
    ```bash
    yarn dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
